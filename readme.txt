== Description ==

Ping.mk додатокот ви овозможува да додадете копче за споделување на вашите написи на македонскиот агрегатор на вести - Ping.mk. Додатокот ви нуди можност да изберете помеѓу три интерактивни копчиња со кои можете да гласате за вестта, додека тоа не беше случај во претходната. Ping.mk копчето по ваш избор може да се покажува на написите и страниците, а позицијата може да биде на почетокот или крајот од написот/страницата и лево или десно.

== Installation ==

1. Прикачете ја „pingmk-share-button“ папката во /wp-content/plugins/.
2. Активирајте го додатокот од менито „Додатоци“.
3. Подесете ги опциите по ваша желба.
4. Алелуја :-)
